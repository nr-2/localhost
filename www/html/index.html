localhost

A single-threaded, epoll-driven HTTP/1.1 server written in Rust.

Static files & directory listing

  • /about.html — another static page
  • /uploads/ — directory listing (autoindex on)

File upload (POST)

Uploaded files are saved under /uploads/ and can be deleted with DELETE /uploads/<filename>.

CGI

  • /cgi-bin/hello.py — Python CGI
  • /cgi-bin/env.sh — Shell CGI

Cookies & sessions

/session — sets a session_id cookie on first visit and counts how many times this session has loaded the page.

Error pages

  • /does-not-exist → 404
  • /old → 301 redirect to /
