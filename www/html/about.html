About

A few details about this server.

This site is served by localhost, a single-threaded, non-blocking HTTP/1.1 server written in Rust on top of a raw epoll event loop.

  • One epoll instance handles every listener, client socket, and CGI pipe.
  • Supports GET, POST, and DELETE, chunked and unchunked request bodies, virtual hosts, cookies/sessions, file uploads, and CGI.
  • Configured via a small nginx-like config file (see conf/default.conf).
